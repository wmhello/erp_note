
# 商品进销存管理系统


## 功能
> 商品信息及种类管理
>
> 进货管理
>
> 销售管理
>
> 打印统计报表
>
> 根据库存进行智能提醒
>
## 如何获取
> 需要演示或者商务合作可以加微信（xpyzwm）

## 使用范围
> 作为库存系统基础模板使用

> 适用于毕业设计及相关产品演示

## 演示网站

>[管理端](https://easy2erp.halian.net)  (用户名：admin 密码:123456)
